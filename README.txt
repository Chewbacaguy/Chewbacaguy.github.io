CRS Tecnologico de Monterrey

Esta pagina web es el sitio donde se desplegara la informacion mas reciente del Consejo de Responsabilidad Social dentro del 
Tecnologico de Monterrey campus Monterrey. La pagina es producto de Santiago Torres Alvarez (@santorres) coordinador de 
computacion del CRS Gestion 2021. 

La pagina esta escrita principalmente con html, css y un poco de javscript.
